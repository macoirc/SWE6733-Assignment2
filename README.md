## Rationale
Microsoft Teams will be used as our verbal means of communication. The group will be able to effectively communicate without delay whilst also monitoring driver-driven code changes through screen share functionality.

Python was chosen as our coding language due to the team members' having the most familiarity and experience with it. This will allow for faster assignment progression and better efficiency when it comes to useful testing and refactoring.

VS Code is an IDE that allows simultaneous live editing and programming with a multitude of coding languages. It adds extra convenience through the possession of its own code coverage tool.

The GitHub repository will be the group's project management and central data keeper. It will host any changes as well as document test coverage and resulting artifacts post the mob programming session.
